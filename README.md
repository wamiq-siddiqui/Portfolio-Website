# Portfolio-Website
A modern and responsive portfolio highlighting my web development skills and showcasing my projects.

## Overview
This is my personal portfolio website built with HTML, CSS, and JavaScript.  
It includes a responsive layout, hamburger menu, form validation, and a dark/light mode toggle.

## Features
- Fully responsive design for all devices  
- Hamburger menu for mobile view  
- Contact form with real-time validation  
- Dark/Light mode switch  
- Smooth transitions and hover effects  

## Live Demo
[View My Portfolio](https://wamiq-siddiqui.github.io/Portfolio-Website/)

## Technologies Used
- HTML5  
- CSS3 (Flexbox, Media Queries)  
- JavaScript (DOM, Validation, Interactivity)
